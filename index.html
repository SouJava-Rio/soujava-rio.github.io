Sobre a SouJava { RIO }

O SouJava é um grupo de usuários, formado por desenvolvedores e evangelistas da tecnologia Java no Brasil, e tem como objetivo fortalecer, expandir e profissionalizar o uso de Java no país. O SouJava é um dos mais ativos e importantes grupos de usuários do mundo, e realiza diversas atividades no Brasil, e ajuda na organização do movimento Java mundial. O SouJava mantém reuniões presenciais, eventos e uma presença online, ajudando a comunidade Java brasileira a aplicar a tecnologia Java em projetos reais.

Quais os benefícios de fazer parte do SouJava { RIO }

Para membros individuais:

  • Estar a par das últimas novidades da plataforma Java.

  • Conhecer as últimas tecnologias através da participação nos eventos do grupo.

  • Ter acesso a especialistas da área ou, se for um especialista numa determinada tecnologia, ter acesso a uma audiência para compartilhar o seu conhecimento.

  • Treinar apresentações num ambiente mais controlado de forma a ganhar experiência para participar em conferências maiores.

Para empresas:

  • Possibilidade de patrocinar um evento e desta forma divulgar algum produto ou serviço aos membros do grupo.

  • Possibilidade de conhecer os melhores profissionais da área podendo contratar os melhores recursos do mercado.
